# NopeID Public Installer

This repository contains the public NopeID installer and installation docs.

```sh
curl -fsSL https://nopeid.com/install.sh | sh
```

Documentation:

- [Installation](docs/install/installation.md)
- [Upgrade](docs/install/upgrade.md)
- [Uninstall](docs/install/uninstall.md)

Common commands after install:

```sh
nopeid --version
sudo nopeid start
```
